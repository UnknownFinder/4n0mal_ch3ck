This is a first-aid-script for checking up for perhaps anomalies, like looking for zombie-processes, frozen cron tasks, unusual processes, ssh anomalies and other thing, that we can name 'f*cking' for the system. You can just run script like 'your@nick~$ sudo sh 4nomal_ch3ck.sh'
-----------------------------------------------------------------------------------------------------------------------------------------------------
P.S - last update was 15.04.2026. Now you can use some options like:
-z check up for zombie proccesses
-c check up for sleeping cron tasks 
-p check up for starnge or unusual ports and coonections
-n check up for network anomalies
-s check up for ssh anomalies
-u check up for critical security updates
-r check up for no-root login
Just write 'your@nick~$ sudo sh 4nomal_ch3ck.sh' with flag, you can combine the flags.
